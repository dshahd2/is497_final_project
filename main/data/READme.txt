INCLUDED IN data

raw_data - data to start with. Includes TKOv3 library to use for alignment and document with links to download the sample runs. Detailed instructions to download sequences are found in project_doc in the main folder. Also includes training sets for BAGEL - CEGv2.txt (essential genes) & NEGv1.txt (non-essential genes).

supplementary_data - data that is given to us by the cited paper (paper where raw data comes from). Includes readcount matrix which can also be used to compare the generated readcount matrix file to. Also includes TKOv3 library - unformatted. Instructions to reformat are found in project_doc in the main folder.
